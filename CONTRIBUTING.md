# Contributing to ADHD Brain

We love contributions! Here's how you can help:

## 🚀 How to Contribute

1.  **Fork** the repository.
2.  **Clone** your fork: `git clone https://github.com/flippinhutt/adhd-brain.git`
3.  **Create a branch** for your feature or fix: `git checkout -b feature/awesome-thing`
4.  **Make your changes**.
5.  **Test your changes**: Run `npm run lint` and verify in the browser.
6.  **Commit your changes**: `git commit -m "Add some awesome thing"`
7.  **Push to your branch**: `git push origin feature/awesome-thing`
8.  **Open a Pull Request**.

## 🛠 Code Style

- Use **TypeScript** for all logic.
- Follow the existing folder structure.
- Keep components small and focused.
- Use **Tailwind CSS** for styling.

## 🐛 Reporting Bugs

If you find a bug, please open an issue with:
- A clear description of the problem.
- Steps to reproduce the issue.
- Expected vs. actual behavior.

## 💡 Feature Requests

Have an idea to make ADHD Brain better? Open an issue and label it as `enhancement`.
